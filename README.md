# Steel Tension Member Stress and Elongation Check

## Objective

This project analyzes a steel tension member subjected to axial tensile loading using Python.

## Calculations

* Cross-sectional area
* Axial stress
* Strain
* Elongation
* Factor of safety
* Safety check

## Required Packages

* numpy
* pandas
* matplotlib
* jupyter

## Run Instructions

Open the notebook using:

jupyter notebook

Then run all notebook cells.
